# English Quiz Bot

Telegram-бот для изучения английского языка через викторины (quiz) с использованием **Telegram Bot API**, **Aiogram** и базы данных SQLite.

---

## 🎯 Цель проекта

Разработать Telegram-бота, который позволяет:

- Проходить английские викторины (тесты с вариантами ответов)  
- Получать оценку и подсказки после ответа  
- Сохранять прогресс пользователя  
- Администрировать вопросы и категории  
- Деплоить бота для постоянной работы  

Проект реализуется поэтапно с применением FSM, базы данных, кнопок, middleware и деплоя.

---

## 📌 Функциональные требования

### 1. Регистрация пользователя
- Автоматическая регистрация при первом запуске
- Сохранение user_id, имени и даты регистрации в БД

### 2. Прохождение викторины
- Пользователь выбирает категорию (например, Vocabulary, Grammar, Idioms)  
- Вопросы выводятся с 4 вариантами ответа (Inline кнопки)  
- FSM шаги: вопрос → выбор ответа → результат → следующий вопрос  

### 3. Подсказки и результаты
- Подсказка по каждому вопросу (если пользователь ошибся)  
- Итоговый результат викторины  
- Сохранение статистики: количество правильных и неправильных ответов  

### 4. Админ-панель
- Доступ только для admin_id  
- Добавление / удаление вопросов  
- Просмотр статистики пользователей  
- Управление категориями викторин  

### 5. Middleware
- Логирование действий пользователя  
- Проверка регистрации  
- Ограничение доступа к админ-командам  

### 6. Деплой
- Бот работает 24/7 на Heroku или VPS
