# Database Structure — English Quiz Bot

## 1️⃣ Таблица `users`

| Поле       | Тип         | Описание                                 |
|------------|------------|------------------------------------------|
| id         | INTEGER PK | Уникальный идентификатор пользователя   |
| user_id    | INTEGER    | Telegram user_id                         |
| name       | TEXT       | Имя пользователя                        |
| created_at | DATETIME   | Дата регистрации пользователя            |

**Примечания:**  
- Сохраняется при первом запуске бота (`/start`)  
- Связь с таблицей `user_stats` для хранения результатов викторин  

---

## 2️⃣ Таблица `categories`

| Поле       | Тип         | Описание                              |
|------------|------------|---------------------------------------|
| id         | INTEGER PK | Уникальный идентификатор категории   |
| title      | TEXT       | Название категории (Vocabulary, Grammar, etc.) |
| created_at | DATETIME   | Дата добавления категории            |

**Примечания:**  
- Категории используются для группировки вопросов  

---

## 3️⃣ Таблица `questions`

| Поле       | Тип         | Описание                                 |
|------------|------------|------------------------------------------|
| id         | INTEGER PK | Уникальный идентификатор вопроса        |
| category_id| INTEGER    | Ссылка на таблицу `categories`          |
| question   | TEXT       | Текст вопроса                            |
| option_a   | TEXT       | Вариант A                                |
| option_b   | TEXT       | Вариант B                                |
| option_c   | TEXT       | Вариант C                                |
| option_d   | TEXT       | Вариант D                                |
| correct    | TEXT       | Правильный вариант (A/B/C/D)            |
| hint       | TEXT       | Подсказка для вопроса (опционально)     |
| created_at | DATETIME   | Дата добавления вопроса                  |

**Примечания:**  
- Вопросы связаны с категориями через `category_id`  

---

## 4️⃣ Таблица `user_stats`

| Поле        | Тип         | Описание                               |
|-------------|------------|----------------------------------------|
| id          | INTEGER PK | Уникальный идентификатор записи        |
| user_id     | INTEGER    | Ссылка на таблицу `users`              |
| category_id | INTEGER    | Ссылка на таблицу `categories`         |
| correct     | INTEGER    | Количество правильных ответов          |
| incorrect   | INTEGER    | Количество неправильных ответов        |
| last_attempt| DATETIME   | Дата и время последней викторины       |

**Примечания:**  
- Позволяет отслеживать прогресс пользователя по каждой категории
