- [ ] Подключить токен через `.env`
- [ ] Реализовать команды: `/start`, `/help`, `/about_bot`

**Результат:**

## Этап 2 — Кнопки
- [ ] Создать Reply клавиатуру (Главное меню)
- [ ] Inline кнопки для выбора категорий викторины
- [ ] Inline кнопки для выбора вариантов ответа на вопрос
- [ ] Кнопка «⬅ Назад» для возврата в главное меню  

**Reply Keyboard (Главное меню):**
- 📝 Start Quiz — начать викторину  
- 📊 My Stats — мои результаты  
- ⚙ Admin — админ-панель (только admin)  

**Inline Keyboard (при выборе вопроса):**
- Варианты ответа A, B, C, D  
- ⬅ Назад — вернуться в выбор категории

- [ ] Реализовать команду `/cancel` для выхода из викторины  
- [ ] Сохранять прогресс пользователя  

**Результат:**
